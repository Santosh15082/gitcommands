These file contains basic git commands
1. git init
Purpose: Initializes a new Git repository in the current directory.
Usage: git init [Image of a terminal window showing the git init command being executed and creating a new Git repository]
2. git clone
Purpose: Creates a local copy of a remote Git repository.
Usage: git clone <remote_repository_url> [Image of a terminal window showing the git clone command being used to clone a GitHub repository]
3. git add
Purpose: Stages changes in your working directory to be committed.
Usage: git add <filename> or git add . (to stage all changes) [Image of a terminal window showing the git add command being used to stage a modified file]
4. git commit
Purpose: Commits staged changes to the local repository.
Usage: git commit -m "Commit message" [Image of a terminal window showing the git commit command being used to commit staged changes with a message]
5. git status
Purpose: Shows the current state of the working directory and staging area.
Usage: git status [Image of a terminal window showing the git status command output, indicating the current branch, modified files, and staged files]
6. git diff
Purpose: Compares changes between files or commits.
Usage: git diff <filename> or git diff HEAD [Image of a terminal window showing the git diff command being used to compare the current working directory with the previous commit]
7. git log
Purpose: Displays the commit history of the repository.
Usage: git log [Image of a terminal window showing the git log command output, displaying a list of commits with their hashes, author information, and commit messages]
8. git branch
Purpose: Creates, lists, or deletes branches.
Usage: git branch <branch_name> (to create a branch) git branch (to list branches) git branch -d <branch_name> (to delete a branch) [Image of a terminal window showing the git branch command being used to create and list branches]
9. git checkout
Purpose: Switches to a different branch.
Usage: git checkout <branch_name> [Image of a terminal window showing the git checkout command being used to switch to a different branch]
10. git merge
Purpose: Merges changes from one branch into another.
Usage: git merge <branch_name> [Image of a terminal window showing the git merge command being used to merge changes from one branch into another]
11. git push
Purpose: Pushes local commits to a remote repository.
Usage: git push <remote_name> <branch_name> [Image of a terminal window showing the git push command being used to push local commits to a remote repository]
12. git pull
Purpose: Fetches and merges changes from a remote repository.
Usage: git pull <remote_name> <branch_name> [Image of a terminal window showing the git pull command being used to fetch and merge changes from a remote repository]
